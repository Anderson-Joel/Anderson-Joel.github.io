The skewX() Method

The skewX() method skews an element along the X-axis by the given angle. Make it negative, it will skew the X-axis the other direction.

This div element is skewed 10 degrees along the X-axis.


The skewY() Method

The skewY() method skews an element along the Y-axis by the given angle. Make it negative, it will skew the Y-axis the other direction.

The skew() Method

The skew() method skews an element into a given angle. Make it negative, it will skew the axis the other directions.

This a normal div element.
This div element is skewed 20 degrees along the X-axis, and -10 degrees along the Y-axis.


The matrix() Method

The matrix() method combines all the 2D transform methods into one. The parameters are as follow: matrix(scaleX(),skewY(),skewX(),scaleY(),translateX(),translateY()).
